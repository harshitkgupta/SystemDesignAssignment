# 
## Setup
`/bin/setup`

## Run Full Test Suite(Ruby Specs)
`bin/run_functional_specs`

## Run Application in CLI mode
`bin/parking_lot`

## Run Application using data from file
`bin/parking_lot <file-path>`

## Environment Description
Language- java

Test Runner- junit

Build tool- gradle

Spec and Spec Runner- Ruby

# Clean project
`./gradlew clean`

# Build project
`./gradlew build`

# Run junit test
`./gradlew test`
